# Exercícios Trybe - Módulos e Projetos

Bem-vindo ao meu repositório de exercícios e projetos desenvolvidos durante o curso da **Trybe**!  
Aqui você encontra todo o conteúdo prático que realizei ao longo da formação Full-Stack.

## Sobre o Repositório

Este repositório contém:
- Exercícios individuais de cada bloco e módulo
- Projetos de fixação e avaliação
- Clones e desafios front-end (HTML, CSS, JavaScript, React, etc.)
- Algoritmos e lógica de programação
- Back-end (Node.js, Express, bancos de dados, etc.)
- Testes unitários, Git/GitHub e boas práticas

**Objetivo principal**: Documentar minha evolução, praticar conceitos e ter um portfólio organizado para consultas futuras e processos seletivos.
